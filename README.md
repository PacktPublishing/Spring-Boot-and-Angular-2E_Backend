# 🛠️ Part 2: Backend Development – Spring Boot Microservices

## Introduction

This repository contains all backend code, resources, and configuration for **Part 2: Backend Development** of the book "Spring Boot and Angular – Second Edition". Here, you will find a complete microservices architecture built with Spring Boot, covering everything from service setup, database integration, API gateway, security, observability, and production readiness. Each chapter is organized in its own folder, with a dedicated README to guide you through the concepts, code, and setup for that chapter.

---

## 📚 Chapters Covered

1. Spring Boot and Angular – Understand the Big Picture and Kickstart the Environment
2. Getting Started With Microservices Using Spring Boot
3. Setting up your development AI Assistant
4. Setting up Database and Repositories using Spring Data JPA
5. Building Application Services and APIs with Spring
6. Developing Service Discovery and API Gateway using Spring Cloud
7. Documenting APIs with OpenAPI and logging the events
8. Securing microservice using Spring Boot Security and JWT
9. Reactive programming using Spring WebFlux
10. Building and Packaging the Spring Boot backend
